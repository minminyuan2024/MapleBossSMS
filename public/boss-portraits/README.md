# Boss portrait assets

Run from the **project root**:

```bash
node scripts/download-boss-portraits.cjs
```

This downloads thumbnails from [MapleStory Wiki](https://maplestorywiki.net/w/Bosses) (same boss pages as the Major Bosses list) into this folder and updates `src/data/boss-portrait-local.json`.

Attribute MapleStory Wiki; images are typically **CC BY-SA**.
